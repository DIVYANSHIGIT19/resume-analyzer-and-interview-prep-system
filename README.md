# 🚀 AI Resume Analyzer & Interview Prep System

## Overview

AI Resume Analyzer & Interview Prep System is an intelligent recruitment assistance platform that helps candidates optimize their resumes and prepare for interviews.

The application uses Natural Language Processing (NLP), Machine Learning techniques, and data analytics to compare resumes with job descriptions, identify missing skills, generate interview questions, evaluate responses, and track performance through interactive dashboards.

---

## Features

### 📄 Resume Analysis

* Upload resume in PDF format
* Compare resume with Job Description
* TF-IDF based similarity matching
* Resume scoring system
* Missing keyword detection

### 🧠 Skill Detection

* Automatically extracts technical skills from resumes
* Supports popular programming and data science technologies

### 🎯 Interview Question Generator

* Generates skill-based interview questions
* Personalized according to detected skills

### 🧠 Practice Mode

* Answer generated questions
* Receive instant evaluation score
* Track interview preparation progress

### 📊 Analytics Dashboard

* Resume ranking leaderboard
* Practice performance ranking
* Interactive visualizations
